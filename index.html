Proyectos

Proyecto de Reseñas de peliculas

Este sistema es una aplicación de reseñas de películas que incorpora un flujo de moderación para garantizar la calidad del contenido visible públicamente. Los usuarios pueden crear, leer, actualizar y eliminar reseñas de películas, mientras que los moderadores tienen la capacidad de aprobar o rechazar dichas reseñas. Solo las reseñas aprobadas se muestran al público y se utilizan para calcular el promedio de puntuación de cada película.

Proyecto de termometro para site o hogar

Aplicación web diseñada para medir y controlar la temperatura y la humedad del entorno donde se instale el sistema. Utiliza un dispositivo Arduino equipado con sensores para realizar las mediciones en tiempo real. El sistema se monta fácilmente en una caja diseñada para facilitar su instalación en el sitio deseado. Además, permite configurar remotamente los parámetros de temperatura a través de la aplicación web, brindando un control climático eficiente y accesible desde cualquier ubicación.

Proyecto de localizacion de Autobuses

Es una aplicación web que permite a los usuarios visualizar en tiempo real la ubicación de autobuses en un mapa. Proyecto de Librería Web

La librería web es una aplicación que permite a los usuarios explorar un catálogo de libros de manera interactiva y sencilla. Los usuarios pueden buscar libros por título, autor o género, ver detalles de cada libro como sinopsis, año de publicación y número de páginas, y agregar títulos a un carrito de compras simulado. Además, la aplicación incluye un panel administrativo básico donde se pueden agregar, editar o eliminar libros del catálogo. El proyecto tiene como objetivo practicar la integración de frontend, backend y base de datos, y simular funcionalidades reales de una librería online, mejorando la experiencia del usuario y facilitando la gestión del inventario.
